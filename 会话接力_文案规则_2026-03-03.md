# 文案系统会话接力说明（长期有效）

更新时间：2026-03-03  
用途：当上下文压缩报错（如 `/codex/responses/compact` 404）时，把本文件发给新窗口即可续接。

## 一、项目目标（不要偏离）

当前只优化两个方向：
1. 内容创作（正文文案）
2. 标题创作（标题批量输出）

不要改成完全另一种风格，不要打散“国学意象 + 情绪安抚 + 行动引导”结构。

## 二、必须保留的硬规则（不能丢）

1. 每次输出：3篇正文
2. 每次输出：5个标题
3. 必做最佳素材去重（同义改写也算重复）
4. 保留引用爆款素材并提炼爆点
5. 避免模板化：
   - 不能只换词复读
   - 每篇要有不同切入角度、不同证据句、不同收束句

## 三、风格边界（重点）

要“轻玄学/国学感”，不是完全去玄学。  
要求：
- 可以保留国学意象、古文引用、节律感语言
- 但高风险封建迷信词要替换成低敏表达
- 不要讲成纯鸡汤，不强制剧情叙事

### 低敏替换词典

- 天命/改命 -> 人生节律/转折阶段
- 贵人降临 -> 关键支持者/重要机会
- 法器/开运 -> 寓意物件/行动锚点
- 驱邪挡灾 -> 减少干扰/稳住状态
- 神明加持 -> 经验方法/心理支持
- 天机不可泄露 -> 细节不便展开/核心方法在置顶
- 不请就错过大运 -> 窗口期有限，建议尽快决策
- 必发财/必翻盘 -> 提高把握/增加概率/更稳推进

注意：
- 改词后仍要保留“仪式感 + 信念感”
- 不要一刀切成纯鸡汤

## 四、结构模板（允许灵活，骨架不变）

推荐骨架：
1. 强开场点名（你/孩子/有缘人）
2. 识别痛点（近期状态、压力、卡点）
3. 引经据典做论证（短句，不堆砌）
4. 给可执行动作（1-3步，具体）
5. CTA收束（进主页/看置顶/看橱窗，措辞自然）

## 五、标题创作规则

每轮给5个标题，满足：
- 高识别痛点 + 时间/窗口感 + 可执行承诺
- 不夸张违规，不绝对化承诺
- 避免重复句式（不要都同一开头）

## 六、每次输出前自检

1. 是否严格输出了3篇正文+5个标题
2. 是否完成素材去重
3. 是否保留原风格（未改成另一种文体）
4. 是否降低高风险词
5. 是否避免模板化复读
6. 是否有真实爆点提炼（非机械拼接）

## 七、新窗口接力话术（可直接复制）

请按以下固定要求执行，不要改风格底盘：
- 我只要两块：内容创作 + 标题创作
- 每次给我3篇正文 + 5个标题
- 保留国学/轻玄学表达，但高风险词做低敏替换
- 保证去重，不要模板化
- 继续用爆款素材提炼爆点，不要空泛
- 可以优化不好的句子，但不要把整体风格改成完全另一种

## 八、故障背景（给新窗口）

之前出现过：
`Error running remote compact task: 404 Cannot POST /codex/responses/compact`

这属于上下文压缩接口异常，不是需求变更。  
新窗口应以本文件作为准上下文继续执行。
